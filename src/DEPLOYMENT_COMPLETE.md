## 🔑 Key Things to Know

### Apple App Store
- Review takes 24-48 hours (usually)
- Be specific in review notes
- Have demo account ready
- Create once, release anytime
- Can schedule release date

### Google Play
- Review takes 2-4 hours (much faster!)
- Start at 5% rollout (safer)
- Expand gradually (5% → 25% → 50% → 100%)
- Monitor for 24h between each
- Stop rollout if crash rate spikes

### Both Platforms
- Can't change code after submission
- Must increment version for update
- Test thoroughly before submitting
- Have support email monitored
- Respond to reviews quickly

---

## 💡 Pro Tips

1. **Start with TestFlight**
   - Catch bugs before App Store
   - Get real user feedback
   - Fix critical issues early

2. **Use Staged Rollout on Android**
   - Don't go 100% immediately
   - Start 5%, expand every 24h
   - Safe way to catch issues

3. **Monitor Daily**
   - Check crash reports hourly (first week)
   - Read reviews immediately
   - Respond to 1-star reviews
   - Fix top crashes urgently

4. **Plan Updates**
   - Collect feedback first week
   - Plan v1.1 features
   - Prioritize bug fixes
   - Release updates regularly
